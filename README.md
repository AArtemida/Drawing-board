## 基础功能
1. 菜单：新建画布；导入本地图片；保存到本地
2. 工具：画笔；橡皮；吸色器；画矩形;画圆形；加文字；填充图形颜色；
3. 画笔：修改画笔大小
4. 色盘：支持修改rgb和手动输入色值修改颜色；支持修改画笔透明度
5. 橡皮：修改橡皮大小和透明度
6. 隐藏工具栏
