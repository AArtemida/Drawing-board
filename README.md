## 基础功能
1. 菜单：新建图；导入本地图；保存到本地
2. 工具：画笔；橡皮；吸色器；
3. 画笔：修改画笔大小
4. 色盘：支持修改rgb和手动输入色值修改颜色
5. 橡皮：修改橡皮大小
6. 隐藏工具栏
